Laptop setup

This is ideally for developers, but great for analysts too.

- Node

As you'll be switching between clients it's highly recommended you setup nvm. Installing this will mean you will not need to install node - as it's managed via nvm https://github.com/nvm-sh/nvm

- Homebrew (Handy for Analysts)

Homebrew is a package manager for mac. It allows easy installs of most applications. I usually recommend searching for the package via homebrew before downloading it and manually installing it.

Try it out by installing AWS CLI

brew install awscli

or install vscode

brew cask install visual-studio-code

Homebrew usually manages command line packages (git, imagemagick, speedtest-cli), whereas casks are installer packages (slack, chrome).

One note with homebrew it's never recommended to manage engines or version managers with it. (Don't use it with pyenv, rvm, nvm, etc)

htps://brew.sh
